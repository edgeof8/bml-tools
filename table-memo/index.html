table-memo

One click. Any HTML table. Clean Markdown in your clipboard.

Drag this button to your bookmarks bar. Then visit any page with a table and click it.

🗃️ table-memo

Or click to copy the bookmarklet code, then create a new bookmark and paste it as the URL.

How it works

1
Navigate to any page with an HTML table — Wikipedia, financial data, comparison charts.
2
Click table-memo — it finds the most data-rich table automatically.
3
Paste anywhere — clean Markdown table is in your clipboard.

Output example

| Element | Symbol | Atomic Number | Weight |
| --- | --- | --- | --- |
| Hydrogen | H | 1 | 1.008 |
| Helium | He | 2 | 4.003 |
| Lithium | Li | 3 | 6.941 |
